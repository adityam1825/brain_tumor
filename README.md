**Brain Tumor Detection Using CNN**

An AI-powered web application for detecting and classifying brain tumors from MRI scans using Convolutional Neural Networks (CNN). The system provides automated prediction, tumor visualization, and an easy-to-use interface for medical image analysis.

🧠 How It Works
User uploads an MRI brain image.
The image is preprocessed:
Resizing
Normalization
Enhancement
CNN extracts image features.
Model predicts tumor type.
Heatmap visualization highlights affected region.
Results are displayed on the web interface.
🚀 Installation & Setup

1️⃣ Clone Repository
git clone https://github.com/adityam1825/brain_tumor.git
cd brain_tumor

2️⃣ Create Virtual Environment
python -m venv venv

3️⃣ Activate Environment
Windows
venv\Scripts\activate
Linux/Mac
source venv/bin/activate

4️⃣ Install Dependencies
pip install -r requirements.txt

5️⃣ Run Application
python app.py
